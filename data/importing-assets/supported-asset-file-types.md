# Supported Asset File Types

The following are the file types that can be annotated on Ango Hub:

| **Category**    | **Supported File Types**     |
| --------------- | ---------------------------- |
| Audio           | .mp3, .wav                   |
| Image           | .jpg, .jpeg, .png .tif .tiff |
| Video           | .mp4, .ogg, .webm, .mov      |
| Text            | .txt .md                     |
| Document        | .pdf                         |
| Medical Imaging | .dcm, .nrrd                  |
